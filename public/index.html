Currency Rates API

GET Currency Rates

http://localhost:4000/api/rates?base=CAD&currency=USD,SEK

Query Parameters

Parameters Description
base home currency rates (i.e. CAD)
currency specific exchange rates(i.e. USD, SEK).

Response


  {
      "results": {
          "base": "CZK",
          "date": "2020-11-17",
